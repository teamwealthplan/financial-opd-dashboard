# Financial OPD Dashboard · Team Wealth Plan

เครื่องมือวิเคราะห์และวางแผนการเงินส่วนบุคคล (digital OPD card) สำหรับงาน private wealth planning

เป็นเว็บ **ไฟล์เดียว** (`index.html`) ไม่ต้อง build ไม่ต้องมี server — ลูกค้ากรอกข้อมูลการเงินของตัวเองครั้งเดียว
ระบบจะคำนวณ **คะแนนสุขภาพการเงิน (0–100)**, อัตราส่วนสำคัญ 5 ตัว, งบดุล/กระแสเงินสด และ **คำแนะนำ** ให้อัตโนมัติ

## ฟีเจอร์
- เริ่มต้นเป็นหน้าว่างให้ลูกค้ากรอกเอง + ไอคอน **ⓘ** อธิบายทุกช่อง (เลื่อนเมาส์ไปชี้)
- Dashboard นำด้วย **คะแนน + รายละเอียดสุขภาพการเงินทั้งหมด** (ไม่มี Profile Card)
- 5 อัตราส่วน: สภาพคล่อง · หนี้/สินทรัพย์ · ภาระชำระหนี้/รายได้ · ออม-ลงทุน/รายได้ · สินทรัพย์ลงทุน/มั่งคั่ง
- **เลือกปี + เปรียบเทียบหลายปี** (year-over-year delta + กราฟแนวโน้ม)
- บันทึกอัตโนมัติในเครื่อง (localStorage) · ส่งออก PDF ได้ (ปุ่มพิมพ์)
- ปุ่ม "ดูข้อมูลตัวอย่าง" โหลดเคสตัวอย่าง (อ้างอิง OPD Card 2569)

## Deploy ขึ้น GitHub Pages
1. สร้าง repo แล้ว push ไฟล์ `index.html` (ไว้ที่ root)
2. ไปที่ **Settings → Pages → Build and deployment**
3. Source: **Deploy from a branch** · Branch: `main` · Folder: `/ (root)` → Save
4. รอสักครู่ เว็บจะอยู่ที่ `https://<username>.github.io/<repo>/`

> ผู้เข้าเว็บใหม่ทุกคนจะเริ่มจากหน้าว่าง (ข้อมูลเก็บเฉพาะในเครื่องของแต่ละคน ไม่ส่งออกที่ไหน)

## ใช้งานในเครื่อง
เปิด `index.html` ด้วยเบราว์เซอร์ได้เลย หรือรันเป็น static server:
```bash
python3 -m http.server 4173   # แล้วเปิด http://localhost:4173
```
